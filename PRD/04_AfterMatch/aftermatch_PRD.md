## 1. 제품 개요

### 한 줄 요약
데이트 직후 5분 디브리핑 — 에너지 점수·레드플래그·감정 기록 → 다음 만남 전 AI 자동 요약 → 3번째 데이트부터 내 연애 패턴 분석.

### 해결하는 문제
r/dating_advice 단골 질문: "두 번째 만남에서 내가 왜 ick 느꼈는지 모르겠어요." 데이트 후 기억은 3일이면 흐릿해지고, 2~5명을 동시에 만나는 현대 데이터들은 노션·메모앱에 분산 기록한다. **데이트 특화 메타인지 도구가 존재하지 않는다.**

### 핵심 가치
- **기억 보존**: 데이트 직후 5분 기록 → 2번째 만남 전 "지난번 요약" AI 자동 발송
- **감정 패턴 발견**: 3회+ 기록 후 AI가 "당신은 에너지 다운 상대와 2번째 만남에서 항상 ick를 느낀다" 같은 패턴 분석
- **라인업 관리**: 여러 명을 동시에 만날 때도 헷갈리지 않는 상대방별 진행 상태 보드

## 2. 사용자

### 주요 사용자
- **누구**: 25~38세, 데이팅앱 또는 소개팅으로 적극적으로 만나는 한국인 (동시 2~5명)
- **상황**: 데이트를 마치고 집으로 돌아가는 지하철 안, 또는 귀가 직후
- **목표**: "이 사람이 괜찮은 건지 아닌지 명확하게 정리하고 싶다"

### 핵심 타겟 세그먼트
| 세그먼트 | 특징 |
|---------|------|
| 진지한 데이터 (여성) | 데이팅코치 고객층과 겹침. 자기 패턴 분석 수요 높음 |
| 활발한 데이터 (남성) | 여러 명 동시 관리 필요. 라인업 보드 핵심 사용 층 |
| 솔로 탈출 의지파 | 반복 실패 패턴을 직접 찾고 싶어하는 층 |

### 사용자 시나리오
1. 3번째 소개팅 후 집으로 돌아가는 길, 앱 열어 에너지 7점 입력
2. 레드플래그 체크: "대화 독점" 선택 + "결제할 때 소극적이었음" 자유 메모
3. 다음 주 4번째 만남 전날, 알림: "지난 만남 요약: 대화 독점, 에너지 7점, 그린 3개"
4. 7번의 기록 후 AI 인사이트: "당신은 에너지 7 이하인 상대는 3번째 만남에서 항상 종료를 선택했습니다"
5. ₩9,900 구독 중 → 무제한 상대방 등록 + 패턴 분석 이용

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름 — 구독 전환
```
첫 데이트 후 기록 (무료) → 2번째 기록 (무료) → 3번째 기록 구독 유도
→ "패턴 분석은 구독 후 가능" → ₩9,900 결제 → 무제한 + 패턴 분석
```

### 핵심 흐름 — 재방문
```
다음 데이트 예정 → 자동 요약 알림 → 앱 열기 → 전 데이트 내용 확인
→ 데이트 후 새 기록 → 패턴 업데이트
```

### 상세 흐름
1. **랜딩**: "데이트 후 5분, 당신의 연애 패턴을 찾아드립니다" (SEO: 연애 패턴, 소개팅 기록)
2. **상대방 등록**: 닉네임 + 만난 플랫폼(틴더/소개팅/직접 등) 입력
3. **디브리핑**: 만남 직후 → 에너지(1~10) → 레드/그린플래그 선택 → 자유 메모
4. **라인업 보드**: 등록된 모든 상대방 카드 뷰. 마지막 만남일 + 에너지 트렌드 아이콘
5. **다음 만남 전 요약**: "민준 씨 4번째 만남 전 — 지난번: 에너지 7, 레드 1개, 그린 2개. AI 조언: 오늘은 먼저 계획을 물어보세요"

## 5. 성공 기준

- [ ] Phase 1 종료 시 유료 구독자 50명/월 (MRR ₩495,000)
- [ ] 무료 3회 → 유료 전환율 25% 이상
- [ ] 기록 작성까지 평균 소요시간 5분 이하
- [ ] 월 평균 데이트 기록 수 8회 이상 (재방문 지표)

## 6. 안 만드는 것 (Out of Scope — Phase 1)

- AI 패턴 분석 — 이유: Phase 2, 데이터 3회+ 누적 필요
- 음성 덤프 — 이유: Phase 2
- 푸시 알림 — 이유: Phase 2 (웹 푸시 구현 필요)
- 상대방과 공유 / 매칭 기능 — 이유: 프라이버시 핵심 가치와 충돌
- AI 채팅 조언 ("이렇게 답장해요") — 이유: 다른 앱(RIZZ)과 겹침, 포지션 혼선
