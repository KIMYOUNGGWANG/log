## 핵심 제약사항

> **음성 AI 원가 주의**: ElevenLabs Conversational AI는 $0.10~0.12/분.
> ₩24,900·$24.99/월 음성 플랜은 월 120분 캡 필수. Phase 1은 텍스트 롤플레이만으로 PMF 검증 후 음성 추가.

> **의료/CBT 표현 주의**: UI에서 "치료", "진단", "치료사" 표현 직접 사용 금지.
> "연습 공간", "불안 감소 훈련", "CBT 기반 접근" 수준으로 제한.

> **북미 시나리오 완전 현지화 필수**: 한국어 시나리오를 번역하지 말고 북미 문화 맞춤으로 새로 제작.
> 한국의 "회식" = 북미의 "networking event". 맥락이 다름.
