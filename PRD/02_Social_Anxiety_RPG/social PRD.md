## 1. 제품 개요

### 한 줄 요약
CBT 노출치료를 게임처럼 — AI가 카페 직원·면접관·소개팅 상대를 연기하고, 유저는 실제처럼 연습해 사회불안을 단계적으로 극복한다.

### 해결하는 문제
한국 성인 4명 중 1명이 경험하는 사회불안 장애. CBT 상담은 회당 8~15만 원에 6개월 대기. r/socialanxiety · 디시 사불갤에서 반복되는 호소: **"혼자 연습할 곳이 없다."** 텍스트 중심 앱(Joyable, DARE)은 영어 전용이며 실제 대화 느낌이 없다.

### 핵심 가치
- **가상 노출 연습**: 실제 상황을 안전하게 반복 연습. 실패해도 현실 결과 없음
- **단계적 난이도**: 카페 주문(레벨 1) → 회식 자기소개(레벨 3) → 발표/갈등(레벨 5). CBT 노출 위계 원칙 기반
- **불안도 측정**: 연습 전후 불안 점수 기록 → "2주 만에 7점 → 4점" 시각화로 동기 유지

## 2. 사용자

### 주요 사용자
- **누구**: 20~35세, 사회불안 mild~moderate, 임상 미진단 또는 대기 중인 한국인
- **상황**: 내일 회식·발표·면접을 앞두고 혼자 연습하고 싶지만 방법이 없는 순간
- **목표**: "실제 상황 전에 충분히 연습해서 덜 떨고 싶다"

### 핵심 타겟 세그먼트
| 세그먼트 | 특징 |
|---------|------|
| IT/연구직 직장인 | 회식·보고·발표 회피. 구독료 지불 여력 있음 |
| 취업 준비생 | 면접 공포 심각. 반복 연습 수요 높음 |
| 복학생/군 전역자 | 오랜 공백 후 사회 재진입 불안 |

### 사용자 시나리오
1. 내일 팀 회식이 있는 직장인 김씨, "회식 자기소개" 시나리오를 플레이
2. 시작 전 불안도 8점 입력 → AI 동료 캐릭터와 3분 텍스트 대화 완료
3. 종료 후 불안도 5점 입력 → "오늘 -3점! 이번 주 최고 기록" 축하 화면
4. 레벨 3 클리어 → 배지 획득 → 주간 리포트에서 3주간 개선 그래프 확인
5. ₩12,900 구독으로 25개 시나리오 전체 접근

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름 — 구독 전환
```
랜딩 페이지 → 무료 시나리오 선택 → 불안도 입력(시작) → AI와 텍스트 롤플레이
→ 불안도 입력(종료) → 결과 화면 → "더 많은 시나리오 보기" → ₩12,900 구독 → 전체 해금
```

### 핵심 흐름 — 반복 사용
```
오늘 연습할 시나리오 선택 → 플레이 → 불안도 기록
→ 레벨업/배지 확인 → 주간 리포트 확인
```

### 상세 흐름
1. **랜딩**: "사회불안이 있는 당신을 위한 안전한 연습 공간" (SEO: 사회불안 극복, 면접 연습 AI)
2. **카테고리 선택**: 카페/회식/면접/소개팅/전화 중 오늘 연습할 상황 선택
3. **시나리오 선택**: 난이도별 카드 UI. 무료 5개는 잠금 해제 상태
4. **불안도 입력**: 1~10 슬라이더 (캐릭터 표정으로 시각화)
5. **롤플레이**: GPT-4o가 NPC 역할. 채팅 UI. 최대 15턴
6. **종료 & 피드백**: 불안도 재입력 + AI 간단 피드백 ("오늘 회피 없이 끝까지 완료!")
7. **진도 확인**: 레벨업 애니메이션 + 배지 획득 팝업

## 5. 성공 기준

- [ ] Phase 1 종료 시 유료 구독자 50명/월 (매출 ₩645,000)
- [ ] 무료 체험 → 구독 전환율 20% 이상
- [ ] 세션 평균 완료율 70% 이상 (중도 이탈 없이 끝까지)
- [ ] 2주 연속 사용자 불안도 평균 2점 이상 감소

## 6. 안 만드는 것 (Out of Scope — Phase 1)

- AI 음성 롤플레이 — 이유: 원가 리스크($0.10-0.12/분), Phase 2에서 별도 가격 티어로 도입
- 그룹/멀티플레이 모드 — 이유: Phase 3
- 실제 CBT 진단 기능 — 이유: 의료 인허가 리스크
- 커뮤니티/소셜 피드 — 이유: 익명성 필요한 사용자 특성상 복잡도 대비 가치 낮음
- 영어/일본어 버전 — 이유: 한국어 PMF 확인 후
