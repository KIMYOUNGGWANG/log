# 사회불안 노출치료 RPG — PRD (Product Requirements Document)

> 생성일: 2026-05-08
> 생성 도구: Show Me The PRD
> 대상 시장: 한국 (Phase 1a) + 북미 (Phase 1b)

---

## 1. 제품 개요

### 한 줄 요약
CBT 노출치료를 게임처럼 — AI가 카페 직원·면접관·소개팅 상대를 연기하고, 유저는 실제처럼 반복 연습해 사회불안을 단계적으로 극복한다.

### 해결하는 문제

**한국 시장**
한국 성인 4명 중 1명이 경험하는 사회불안 장애. CBT 상담은 회당 8~15만 원에 6개월 대기. r/socialanxiety · 디시 사불갤에서 반복되는 호소: "혼자 연습할 곳이 없다."

**북미 시장**
미국 성인의 약 15%(4천만 명)가 사회불안 장애 경험. CBT 상담 회당 $150~300, 대기 수개월. r/socialanxiety 구독자 440만 명. 영어권 경쟁 앱(Joyable·DARE)은 실제 대화 느낌이 없거나 서비스 종료 상태.

### 핵심 가치
- **가상 노출 연습**: 실제 상황을 안전하게 반복. 실패해도 현실 결과 없음
- **단계적 난이도**: CBT 노출 위계 기반. 레벨 1(카페 주문) → 레벨 5(갈등·발표)
- **세션 메모리 AI**: 이전 연습을 기억하고 패턴을 인식해 맞춤 피드백
- **GAD-7 임상 척도**: 국제 표준 불안 척도로 객관적 개선 수치 확인
- **완전 현지화 시나리오**: 한국·북미 문화 각각 맞춤 25개 시나리오

## 2. 사용자

### 주요 사용자

**한국 세그먼트**
- **누구**: 20~35세, 사회불안 mild~moderate, 임상 미진단 또는 대기 중인 한국인
- **상황**: 내일 회식·발표·면접을 앞두고 혼자 연습하고 싶지만 방법이 없는 순간
- **목표**: "실제 상황 전에 충분히 연습해서 덜 떨고 싶다"

**북미 세그먼트**
- **누구**: 18~35세, 사회불안 mild~moderate, 치료비 부담 또는 대기 중인 영어권 사용자
- **상황**: 네트워킹 이벤트·취업 면접·첫 데이트를 앞두고 연습 공간이 필요한 순간
- **목표**: "실제처럼 연습해서 불안을 낮추고 싶다"

### 핵심 타겟 세그먼트

| 세그먼트 | 한국 | 북미 |
|---------|------|------|
| 직장인 | 회식·보고·발표 회피. 구독료 지불 여력 있음 | 팀 미팅·networking 회피. 구독료 지불 여력 있음 |
| 취업 준비생 | 면접 공포 심각. 반복 연습 수요 높음 | 동일 |
| 재진입자 | 군 전역·복학 후 사회 재진입 불안 | Gap year·이민 후 재진입 불안 |

### 사용자 시나리오

**한국 시나리오 (김씨)**
1. 내일 팀 회식이 있는 직장인 김씨, "회식 자기소개" 시나리오를 선택
2. GAD-7 7점 입력 → AI 동료 캐릭터와 15턴 텍스트 대화 완료
3. GAD-7 4점 입력 → "오늘 -3점! 이번 주 최고 기록" 축하 화면
4. 레벨 3 클리어 → 배지 획득 → 2주간 개선 그래프 확인
5. ₩12,900 구독으로 25개 시나리오 전체 접근

**북미 시나리오 (Alex)**
1. 다음 주 networking event가 있는 직장인 Alex, "Networking Event" 시나리오 선택
2. GAD-7 8점 입력 → AI 낯선 사람 캐릭터와 15턴 텍스트 대화
3. GAD-7 5점 입력 → 세션 메모리 AI: "Last time you struggled with initiating. Today you did great!"
4. 레벨 3 클리어 → 배지 획득
5. $9.99/월 구독으로 25개 영어 시나리오 전체 접근

## 5. 사용자 흐름 (User Flow)

### 핵심 흐름 — 구독 전환
```
랜딩 페이지
  → 언어 자동 감지 (ko/en)
  → 무료 시나리오 선택 (5개)
  → GAD-7 간이 측정 (시작 전)
  → AI와 텍스트 롤플레이 (최대 15턴)
  → GAD-7 재측정 (종료 후)
  → 결과 화면 + AI 피드백
  → "더 많은 시나리오 보기"
  → 구독 결제 (₩12,900 or $9.99)
  → 전체 25개 해금
```

### 핵심 흐름 — 반복 사용 (메모리 AI 포함)
```
오늘 시나리오 선택
  → 세션 메모리 AI 브리핑 ("지난번엔 눈 맞춤에서 막혔어요. 오늘은 의식해보세요.")
  → GAD-7 측정 → 롤플레이 → 재측정
  → 레벨업/배지 확인
  → 주간 GAD-7 트렌드 리포트
```

### 상세 흐름
1. **랜딩**: "A safe space to practice social situations" / "사회적 상황을 안전하게 연습하는 공간"
2. **소셜 로그인**: Google 또는 Apple 1회 로그인
3. **카테고리 선택**: 카페/회식/면접/소개팅/전화 (KO) · Cafe/Networking/Interview/Date/Phone (EN)
4. **시나리오 선택**: 레벨별 카드 UI. 무료 5개는 자물쇠 해제 상태
5. **GAD-7 측정**: 7문항 압축 버전 (3문항 빠른 체크)
6. **롤플레이**: GPT-4o가 NPC 역할. 채팅 UI. 최대 15턴
7. **세션 메모리 반영**: AI가 이전 대화 패턴 기억 후 이번 세션에 반영
8. **종료 & 피드백**: GAD-7 재측정 + AI 간단 피드백 + 개선 시각화
9. **진도 확인**: 레벨업 애니메이션 + 배지 팝업

## 6. 성공 기준

**Phase 1a (한국) 목표**
- [ ] 유료 구독자 50명/월 (매출 ₩645,000)
- [ ] 무료 체험 → 구독 전환율 20% 이상
- [ ] 세션 평균 완료율 70% 이상 (중도 이탈 없이 끝까지)

**Phase 1b (북미) 목표**
- [ ] 유료 구독자 50명/월 (매출 $499.50)
- [ ] 30일 리텐션 10% 이상 (업계 평균 3.3% 대비 3배)
- [ ] r/socialanxiety 커뮤니티 오가닉 언급 5건 이상

## 7. 안 만드는 것 (Out of Scope — Phase 1)

- 음성 롤플레이 — 이유: 원가 리스크($0.10~0.12/분), Phase 2에서 별도 가격 티어로 도입
- 인지 재구성 모듈 — 이유: 롤플레이 PMF 확인 후 Phase 2
- 그룹/멀티플레이 모드 — 이유: Phase 3
- 실제 CBT 진단 기능 — 이유: 의료 인허가 리스크
- 커뮤니티/소셜 피드 — 이유: 익명성 필요한 사용자 특성상 복잡도 대비 가치 낮음
- 일본어 버전 — 이유: 한국+북미 PMF 확인 후
- 코치/치료사 연결 — 이유: Phase 3
