# Phase 2: Voice AI Roleplay Design

## 1. 개요 (Overview)
Phase 1에서 구축한 텍스트 기반 노출 훈련 MVP를 확장하여, **실제 사람과 대화하는 것 같은 음성 AI 대화 훈련**을 도입합니다. 사회불안증 극복을 위한 가장 강력한 훈련 환경을 제공하고, 프리미엄 구독 티어를 신설하여 ARPU(가입자당 평균 수익)를 극대화하는 것이 목표입니다.

## 2. 아키텍처 및 기술 스택 (Architecture & Tech Stack)
* **음성 처리 모델**: `gpt-4o-mini-realtime-preview`
  * 원가 절감(gpt-4o 대비 1/10)과 80% 이상의 높은 마진율 확보
* **통신 방식**: OpenAI Realtime API (WebRTC 기반)
  * 초저지연(~500ms)으로 실제 전화 통화와 같은 몰입감 제공
  * 텍스트 변환(STT/TTS) 병목 없이 오디오 스트림 직접 송수신

## 3. UX/UI 설계 (User Experience)
* **모드 선택**: 시나리오 진입 전 '텍스트 훈련'과 '음성 훈련(통화 모드)' 선택 분기
* **통화 모드 UI**:
  * 전화 통화 화면과 유사한 인터페이스 제공 (상대방 아바타, 통화 타이머, 종료 버튼)
  * **마이크 제어**: 기본 오픈 마이크 (VAD - Voice Activity Detection). 사용자가 침묵할 때를 대비해 하단에 '음소거(Mute)' 버튼 배치
* **시간 제한 게이지**: 프리미엄 유저 대상 '남은 통화 시간(예: 112분/120분)' 시각화 제공

## 4. 과금 및 비즈니스 모델 (Monetization & Limits)
* **신규 구독 티어**: Pro Plan 신설 (₩24,900 / 월)
  * 기존 Premium Plan(₩12,900/월)은 텍스트 무제한
* **사용량 제한 (Cap)**: Pro 구독자당 **월 120분**의 음성 통화 시간 제한
* **무료 체험**: 신규 가입자 전원에게 결제 유도를 위한 **3분 무료 음성 통화** 제공
* **유닛 이코노믹스 (120분 전부 소진 시)**:
  * 매출: ₩24,900
  * 비용: 약 ₩3,980 (결제수수료, 서버, API 원가 포함)
  * 순수익: 약 ₩20,920 (마진율 84%)

## 5. 데이터베이스 변경 사항 (Database Updates)
* `subscriptions` 또는 `profiles` 테이블에 `voice_minutes_used` (Int) 컬럼 추가
* 음성 세션 로그를 기록하기 위한 `VoiceSession` 테이블 신설 (사용 분(minute) 추적)
