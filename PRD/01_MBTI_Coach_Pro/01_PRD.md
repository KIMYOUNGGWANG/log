## 1. 제품 개요

### 한 줄 요약
MBTI·애착유형·에니어그램 결과를 받은 직후의 "그래서 나 어떻게 살아야 해?" 질문에 AI가 90일 행동 계획으로 답해주는 서비스.

### 해결하는 문제
전 세계 10억+ 명이 16Personalities에서 MBTI 테스트를 받지만, 결과 페이지를 닫으면 아무것도 남지 않는다. "INFP는 이래요"라는 설명 이후 실제로 무엇을 바꿔야 하는지 알려주는 서비스는 존재하지 않는다.

### 핵심 가치
- **테스트 → 행동** 전환: 성격 유형 설명에서 끝나지 않고, 오늘부터 90일간 매일 1분씩 실천할 수 있는 구체적 미션 제공
- **한국 특화**: 한국인이 집착하는 MBTI + 애착유형 + 에니어그램을 통합해 다른 어떤 앱도 제공 못 하는 "나 종합 분석"
- **친구/연인 호환성**: "INFP랑 ENTJ 잘 맞나요?" → 구체적 갈등 패턴 + 대화 가이드 PDF 생성

## 2. 사용자

### 주요 사용자
- **누구**: 18~32세 한국 여성 (MBTI 관심도 압도적 1위 세그먼트), 글로벌 확장 시 영어권 Gen Z
- **상황**: 16Personalities·MBTI 관련 유튜브·SNS를 본 직후, 또는 친구와 MBTI 이야기를 나눈 직후
- **목표**: "내 유형을 알았으니 이제 뭘 어떻게 해야 더 나은 사람이 될 수 있을까"

### 사용자 시나리오
1. 유저가 "INFP 특징" 유튜브를 보고 공감하며 더 알고 싶어 검색
2. MBTI Coach Pro의 무료 통합 테스트를 받고 INFP + 불안형 애착 + 에니어그램 4번 결과 획득
3. ₩9,900로 AI 심층 리포트를 결제해 "당신이 반복하는 회피 패턴과 90일 변화 로드맵" 수령
4. ₩29,900 코스를 등록해 매일 아침 1분 미션("오늘 불편한 감정 이름 붙이기")을 받음

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름 — 리포트 구매
```
랜딩 페이지 → 무료 테스트 시작 → 28문항 응답 → 유형 결과 화면
→ AI 리포트 미리보기 (20% 공개) → ₩9,900 결제 → 전체 리포트 확인
→ 카카오/인스타 공유
```

### 핵심 흐름 — 90일 코스
```
리포트 구매 후 → "90일 변화 코스" CTA → ₩29,900 결제
→ 오늘의 미션 홈 화면 → 매일 1분 체크인 → AI 피드백 → 진도 확인
```

### 상세 흐름
1. **랜딩**: SEO 최적화 랜딩 ("MBTI 테스트 무료" 키워드 타겟)
2. **테스트**: 28문항 → 결과 유형 계산 (프론트엔드 로직)
3. **리포트 미리보기**: 서버에서 GPT-4o로 리포트 생성 후 20%만 표시
4. **결제**: Toss Payments → webhook으로 결제 확인 → 전체 리포트 unlock
5. **공유**: `og:image` 동적 생성 (Vercel OG) → SNS 공유 → 바이럴

## 6. 안 만드는 것 (Out of Scope — Phase 1)

> AI에게 코드를 시킬 때 이 목록을 함께 공유하세요.

- 90일 코스 콘텐츠 및 진도 UI — 이유: Phase 2, 리포트 PMF 먼저
- 호환성 PDF — 이유: Phase 3, 바이럴 도구는 코어 검증 후
- 브라우저 푸시 알림 — 이유: Phase 3
- 영어 버전 — 이유: 한국어 PMF 확인 후 글로벌
- 소셜 피드 / 커뮤니티 기능 — 이유: 복잡도 대비 초기 가치 낮음
- 실제 MBTI 자격 인증 (MBTI 공식 라이선스) — 이유: 비용 과다, 16Personalities 방식으로 대체
