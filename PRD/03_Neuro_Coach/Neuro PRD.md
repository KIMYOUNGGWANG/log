## 1. 제품 개요

### 한 줄 요약
매일 아침 1분 체크인 → AI가 오늘의 신경계 상태를 읽고 → 호흡법·냉온수·명상 중 최적 루틴 처방 → 14일 동안 번아웃을 단계적으로 회복시킨다.

### 해결하는 문제
Calm·Headspace는 "명상 콘텐츠 라이브러리"일 뿐, 오늘 내 상태에 맞는 루틴을 골라주지 않는다. 번아웃 직장인은 무엇을 해야 할지 모르기 때문에 아무것도 안 한다. **측정 → 처방 → 실행** 3단계가 하나의 앱에 없다.

### 핵심 가치
- **개인화 처방**: 오늘의 스트레스·수면·에너지 입력 → GPT-4o가 3가지 루틴 중 오늘에 맞는 조합을 추천 ("수면 4점, 스트레스 9점 → 지금 당장 4-7-8 호흡 5분 먼저")
- **신경계 조절 특화**: Calm이 다루지 않는 냉온수 루틴, HRV 기반 처방, 번아웃 단계 맞춤 접근
- **14일 회복 커리큘럼**: 번아웃은 하루에 낫지 않는다. 14일 단계별 코치 콘텐츠로 리텐션 구조화

## 2. 사용자

### 주요 사용자
- **누구**: 30~45세 한국 직장인, 번아웃 경험 중 또는 직전, 건강에 관심은 있지만 "뭘 해야 할지 모르는" 사람
- **상황**: 아침에 일어나도 피곤하고, 퇴근 후에도 회복이 안 되는 순간
- **목표**: "명상 앱 켰다가 뭘 해야 할지 몰라 끄는" 패턴 탈피 → 오늘 할 것을 AI가 정해줬으면

### 핵심 타겟 세그먼트
| 세그먼트 | 특징 |
|---------|------|
| IT 직장인 | 수면 부족·고강도 집중력 소모. $19.9 지불 의사 높음 |
| 번아웃 회복 중인 퇴직자 | 공백기 루틴 부재. 14일 코스 구조 선호 |
| 불안장애 경계선 | 명상 앱 효과 불충분 경험. 처방형 앱 선호 |

### 사용자 시나리오
1. 아침 7시, 수면 3점·스트레스 8점 입력
2. AI: "오늘은 교감신경이 과활성화 상태입니다. 박스 브리딩 4분 → 냉수 샤워 20초 순서를 권장합니다"
3. 박스 브리딩 애니메이션 가이드 완료 → "완료! 오늘 이미 30% 회복"
4. 14일 코스 3일차 — "번아웃의 3단계 패턴 인식하기" 읽고 오늘 숙제 완료
5. 주간 그래프: 이번 주 스트레스 평균 7.2 → 지난주 8.5 대비 -1.3

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름 — 구독 전환
```
랜딩 페이지 → 무료 체크인 1회 체험 → AI 처방 미리보기 (흐림 처리)
→ $19.9/월 구독 → 전체 루틴 + 14일 코스 해금
```

### 핵심 흐름 — 매일 루틴
```
알림 → 앱 열기 → 체크인 (1분) → AI 처방 확인
→ 오늘의 루틴 실행 (5~15분) → 완료 기록 → 14일 코스 진도 +1
```

### 상세 흐름
1. **랜딩**: "번아웃, 무엇을 해야 할지 AI가 정해드립니다" (SEO: 번아웃 극복, 신경계 조절)
2. **체크인**: 3개 슬라이더 (스트레스/수면/에너지). 캐릭터 표정으로 시각화
3. **AI 처방**: "오늘 상태 분석 → 추천 루틴 2~3개 + 이유 한 줄"
4. **루틴 실행**: 각 루틴별 타이머/애니메이션/가이드 화면
5. **코스 콘텐츠**: 매일 1개 카드형 코치 메시지 + 오늘의 숙제

## 5. 성공 기준

- [ ] Phase 1 종료 시 유료 구독자 100명/월 (MRR $1,990)
- [ ] 14일 코스 완주율 40% 이상
- [ ] 체크인 연속 7일 달성 비율 50% 이상
- [ ] 2주 사용 후 평균 스트레스 점수 1.5점 이상 감소

## 6. 안 만드는 것 (Out of Scope — Phase 1)

- 웨어러블 HRV 연동 — 이유: Phase 2, 웹 MVP로 먼저 PMF 검증
- 모바일 앱 — 이유: Phase 2
- 오디오 명상 라이브러리 (콘텐츠 과다) — 이유: TTS로 시작, 수요 확인 후 전문 오디오 제작
- B2B 대시보드 — 이유: Phase 3
- 커뮤니티/그룹 기능 — 이유: 개인 회복이 코어, 사회 기능은 복잡도 대비 가치 낮음
